## 启动方式

```bash
# 1. 进入项目目录
cd wujing-platform

# 2. 安装依赖（首次运行）
pip install -r requirements.txt

# 3. 切割敌情图片（首次运行）
python split_images.py

# 4. 启动服务
python app.py
```

启动后访问：http://127.0.0.1:5000

## 功能模块

1. **基本情况输入** - 输入/上传作战基本情况（/situation）
2. **分析研判** - 敌情/我情/战场环境/综合分析（仪表盘内）
3. **定下作战决心** - 智能生成作战决心文书（/decision）
4. **形成构想方案** - 查看战术部署地图（/concept）
5. **拟制作战计划** - 管理作战计划文档（/formulate）
6. **实施作战推演** - 图上/实兵/网上/综合推演（/wargame）
7. **系统管理** - 用户增删查（/admin，仅管理员）
